# Progress

Current status, roadmap, and completed milestones.
`status.md` should be **always included** in every AI session.

## What goes here

- **Status** — what's done, what's in progress, what's next (keep current!)
- **Roadmap** — ordered list of planned features/milestones
- **Changelog** — brief log of what was built in each session

## Key documents

```
status.md     — current state: done, in-progress, next (always include)
roadmap.md    — full ordered plan with dependencies
changelog.md  — session-by-session log of completed work
```

## Guidelines

- Update `status.md` at the end of every development session
- Keep `status.md` under 50 lines — it's always-include context
- `changelog.md` grows over time; only include recent entries in AI context
- Use concrete descriptions: "Riemann tensor from Christoffel + derivatives"
  not "implemented more operations"
