# References

External resources the AI may need for deeper background.

## What goes here

- **Papers** — summaries of key papers with the specific equations/sections relevant
  to the implementation (not full papers — extract what matters)
- **Textbook excerpts** — specific formulas or derivations from MTW, Wald, Carroll, etc.
- **Prior art** — notes on how other tensor/GR libraries handle similar problems
- **Related repositories** — what exists in the ecosystem and how this library differs

## Key documents

```
papers.md              — summaries of referenced papers with relevant equations
textbook_formulas.md   — canonical formulas we implement (with source citations)
prior_art.md           — other tensor/GR libraries and their trade-offs
```

## Guidelines

- Always cite the source (author, title, equation number)
- Extract the specific formula or algorithm — don't summarize vaguely
- Note sign conventions and index ordering used by each source
- Flag where sources disagree (MTW vs Wald sign conventions, etc.)
